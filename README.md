## App Description
A SwiftUI app demonstrating context menu functionality with two tabs:
- **Food Menu**: Long press food items to choose "Spicy" or "Vegetarian"
- **Drink Menu**: Long press drink items to choose "Hot" or "Cold"

## Features
- Two-tab interface (Food & Drinks)
- Context menus activated by long press
- Visual feedback with SF Symbols
- Clean and intuitive UI

## How to Use
1. Switch between Food and Drink tabs
2. Long press any item to open context menu
3. Select an option to display the corresponding icon
4. Icons provide visual confirmation of selections

## Technical Details
- Built with SwiftUI
- Context Menu modifiers
- TabView navigation
- State management for menu selections
